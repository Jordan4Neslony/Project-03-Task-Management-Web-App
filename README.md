# Project-03-Task-Management-Web-App
# Author: Jordan Neslony

# Description:
This website aims to provide users with accurate gifs corresponding to the text answer they submitted in the search bar.
This website provides users with a todo list they can use to add, check off, and remove todos from. Additionally, if they so wish they can fill out the contact form to contact me.

# Functionality: 
- This project features a nav-bar at the top of the screen that is navigatable
 via hyperlinks.
- The main feature of this website is its todo list. Upon typing in a phrase and pressing the submit button it will add a todo to the list of todos. Clicking on the todos will mark them off as completed turning the todos color green and striking through the text. Finally, clicking on the red 'x' button will delete the todo from the list.
- This website features a contact me form as well 
- Additionally this website is accesible for mobile devices and lower screen resoulution.

# List of Container and Presentational Components:

2. App Screen

# Technologies Used: HTML, CSS, SCSS, Bootstrap, React (through Vyte), Babel, Node.js, and Typescript.

1. As a student, I want to keep myself reminded of my tasks, so that I can excel in school.
2. As an employee, I want to have a checklist of my priorities, so that I  can complete my duties at my job effectively.
3. As a parent, I want to have a to-do list, so that I can help my family in the best way I can.
